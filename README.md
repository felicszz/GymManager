# GymManager

Aplicación Android para la gestión integral de un gimnasio.
Permite administrar socios, rutinas de entrenamiento, clases
colectivas y cuotas. Incluye acceso por QR, inicio de sesión
con Firebase Auth, sincronización en la nube y notificaciones push.

## Stack tecnológico
Kotlin, Android Studio, Room (SQLite), Firebase Auth, Firestore, FCM

## Estructura del proyecto
- `app/src/main/java/` — código fuente Kotlin
- `app/src/main/res/` — layouts, drawables y recursos
- `app/src/main/res/layout/` — pantallas XML

## Funcionalidades previstas
- Gestión de socios (alta, baja, modificación)
- Control de acceso por QR
- Gestión de rutinas y ejercicios
- Clases colectivas con horarios y aforo
- Control de cuotas y pagos
- Notificaciones push (FCM)
- Sincronización en la nube (Firestore)
